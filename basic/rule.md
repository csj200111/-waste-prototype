## 개발 룰

모바일 UI 기준으로 구현

기술 스택

frontend: Typescript + React

backend: java + Spring

DB: mySQL

개발 도중 API 키, DB 연동 등 나의 도움이 필요한 경우에는 나에게 필요한 요구사항을 말할 것.

후에 추가 기능 및 확장에 용이하도록 코딩을 할 것.

데이터베이스 연동이나 지도 API 연동을 위한 확장에 용이하도록 코딩을 할 것.

지도 API 는 나중에 키를 입력하기만 하면 바로 연동할 수 있도록 개발을 할 것.

설정 지역을 대한민국 전국 모든 자치구로 설정할 것.

실제 서비스 가능한 수준까지 개발할 것.

지역을 설정했을 경우 해당 지역과 데이터베이스 내의 해당되는 지역과 매핑을 해줄 수 있을 것.

결재 기능은 '결재가 가능하다' 라는 가정하에 겉모습만 구현할 것.

디자인 설계는 현재 프로토타입에서 크게 벗어나지 말 것.

## 기획

대형 폐기물을 버리는 것은 두가지 방법이 있다.

하나는 오프라인

하나는 온라인

오프라인은 거주지에서 정해진 곳에 버리는 방법

비치된 대형폐기물 배출 신고서에 품목, 규격(크기), 수량, 배출 장소 등을 적는다.

그리고 수수료를 현금이나 카드로 결제한다

결제가 끝나면 대형페기문 신고필증(스티커)를 발급받는다.

스티커를 폐기물 잘 보이는 곳에 붙어서, 약속된 날짜와 장소에 내놓으면 된다.

혹은 주변에 마켓이나 편의점에서 스티커를 판매한다.

구청 청소 행정과에 문의하거나 구청 홈페이지에서 대형폐기물 스티커 판매소 리스트를 확인할 수 있다.

해당 상점에 가서 버리려는 물건의 크기를 말하고 가격에 맞는 스티커(납부필증)을 구매한다.

그 후  스티커를 붙여서 배출한다.

온라인 방식은 구청의 행정 시스템을 이용하는 방법이다.

거주하는 자치구청 홈페이지에 들어간다.

대형폐기물 배출신청 메뉴를 찾아 본인 인증 후 배출 품목과 날짜를 입력한다.

수수료를 계좌이체나 카드로 결제한다.

신고필증을 프린터로 인쇄해서 테이프로 붙인다.

우리는 사용자들이 자신이 버리고자 하는 대형폐기물의 수수료가 얼마인지 쉽게 확인하고 더 나아가 대형폐기물을 버리는 과정에서 더욱 편리하고 쉽게 버릴 수 있는 기능을 서비스 한다.

## 오프라인으로 버리길 원하는 사용자

오프라인으로 버릴 때 필요한 것.

각 기능은 한 화면에 몰아넣지 말고 페이지 내 별도의 공간(버튼)을 통해 제공할 것.

필요한 기능

1. 내가 버리는 대형 폐기물이 어떤 종류고 수수료가 얼마가 나올지(지역입력+규격입력)
    
    이건 그냥 수수료가 얼마가 나올지 확인하는 용도.
    
2. 스티커(납부필증) 을 판매하는 곳이 현재 위치 기준 가까운 곳이 어디인지 안내하기. + 혹은 동사무소 같은 시설위치를 안내하기(직접 가서 신청하도록)
3. 운반 대행: 너무 무거운 대형 폐기물은 옮길 수 없으므로 이를 대신 운반해주는 업체를 연결해주는 기능. (지금은 업체 이름과 전화번호를 보이는 정도로만 구현)
4. 지역 재활용 센터(중고 가게) 혹은 자재가 필요한 업체가 가져갈 수 있도록 대형 폐기물 사진을 올리면 필요한 사람이나 업체가 가져갈 수 있도록 하는 역경매 시스템.
5. 만약 수거 업체가 물건을 가져가길 원한다면 수거업체에게 내가 물건을 버린 위치를 지도로 찍을 수 있게 하여 대형폐기물을 가져갈 수 있도록 하는 기능.

사용자가 지자체. 동사무소의 경우

1. 대형 폐기물을 수거해가는 업체와 연결해주는 기능.
2.  수거업체에게 내가 물건을 버린 위치를 지도로 찍을 수 있게 하여 대형폐기물을 가져갈 수 있도록 하는 기능.

## 온라인으로 버리길 원하는 사용자

일반인의 경우

온라인 상으로 지역, 규격을 입력 후 수수료를 결재 후 배출 번호를 발급받고 직접 버리는 것. 버리는건 직접 해야함.

필요한 기능.

1. 자신이 사는 지역, 버리고자 하는 대형폐기물의 종류와 규격을 입력하면 자동으로 해당 대형폐기물의 수수료가 얼마인지 안내. 사용자가 주소를 입력하면 해당 주소가 어느 구/동인지 자동으로 파악, 해당 지역의 수거 업체와 수수료 테이블을 매핑해야 함. 폐기물 종류는 트리 구조의 카테고리로 제공되면 좋을 듯 (가구 → 책상 → 편수) + 키워드 검색 기능
2. 내가 버리는 대형 폐기물의 수수료가 얼마가 나올지 지역별 기준에 따라 제공.
3. 본인 인증 후 수수료를 결제하고 배출번호를 출력해주는 기능. (종이에 이 배출 번호를 크게 적어 붙이라는 문구 필요)
4. 배출 확인증(영수증) 조회: 증빙을 요구하는 경우 모바일 화면으로 보여줄 수 있는 전자 영수증 기능이 필요
5. 운반 대행: 너무 무거운 대형 폐기물은 옮길 수 없으므로 이를 대신 운반해주는 업체를 연결해주는 기능.
6. 만약 수거 업체가 물건을 가져가길 원한다면 수거업체에게 내가 물건을 버린 위치를 지도로 찍을 수 있게 하여 대형폐기물을 가져갈 수 있도록 하는 기능.
7. 마지막 검수 기능: 자신이 찍은 사진이나 입력한 지역 정보, 규격이 정확한지 다시한번 확인 할 수 있는 기능. 그 후 결재 기능 제공
8. 상태 표시: 마이 페이지에 들어가면 자신이 신청한 내역을 볼 수 있다. 
9. 결재 취소 및 환불 신청 기능. 배출 전 이라면 사용자가 직접 취소할 수 있어야 한다.
